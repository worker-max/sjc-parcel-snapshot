# SJC Parcel Snapshot (TY2025)

Failover snapshot of San Joaquin County's public parcel lookup table
(https://app.sjgov.org/ttc/parceladdress.txt) used by the WFW voice-agent
parcel lookup when the live county file is unavailable or truncated.

- sjc_parcels_2025.csv.gz — 274,126 rows, captured 2026-06-29
- Columns: assessment,parcel_dashes,address,city_state_zip
- BillURL derived: https://common3.mptsweb.com/MBC/sanjoaquin/tax/main/{assessment}/2025/0000

Public county record data. Refreshed opportunistically.
